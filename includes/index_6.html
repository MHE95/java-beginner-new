Kommentare

  • Programme werden schnell unübersichtlich.
  • Jeder Programmiert anders, was dazu führt, dass das Programm eines anderen schwer zu verstehen sein kann.
  • Oft ist bei Variablen nicht klar, für was diese stehen.
    • public class Kommentare {
    public static void main(String[] args) {
        int a = 20; 
    }
}
  • Was wird in "a" gespeichert?

Kommentare

  • All diese Probleme werden durch Kommentare gelöst!
    • public class Kommentare {
    public static void main(String[] args) {
        // a ist das Alter der Nutzers
        int a = 20; 
    }
}
  • Dadurch wird der Quellcode verständlicher.
  • Kommentare ändern nichts am Ablauf des Programms. Sie sind nur für den Programmierer zum Verständnis da.

Kommentare - Einzeilig

  • Einzeilige Kommentar beginnen mit "//".
  • Alles, dass nach "//" kommt wird nicht vom Compiler ausgewertet und verändert den Programmfluss nicht.
  • Meistens wird mit einzeiligen Kommentaren die nächste Zeile erklärt.
    • public class EinzeiligerKommentar {
    public static void main(String[] args) { 
        // Das ist ein einzeiliger Kommentar 
        // In der nächsten Zeile wird ein Text ausgegeben 
        System.out.println("Hallo Welt"); 
        // Auch hier steht ein Kommentar 
    }
}

Kommentare - Mehrzeilig

  • >Kommentare über mehrere Zeilen beginnen mit "/*" und enden mit "*/".
  • Eclipse schreibt an den Anfang  jeder Zeile dazwischen ein "*".
  • Wird für längere Kommentare benutzt.
    • public class EinzeiligerKommentar {
    /*
    * Dies ist ein mehrzeiliger Kommentar.
    * Das Programm gibt "Hallo Welt!" auf der Konsole aus.
    */
    public static void main(String[] args) { 
        System.out.println("Hallo Welt"); 
    }
}

Kommentare - Mehrzeilig

public class Rechner { 
    /* 
    * Dieses Programm berechnet die Stunden pro Woche 
    * Das Ergebnis soll am Ende ausgegeben werden 
    */ 
    public static void main(String[] args) { 
        // Anzahl Stunden pro Tag 
        int stunden = 24; 
        // Anzahl Tage pro Woche 
        int tage = 7; 
        // Berechnung Stunden pro Woche 
        int ergebnis = stunden * tage; 
        // Ausgabe: Anzahl Stunden pro Woche 
        System.out.println("Anzahl Stunden pro Woche: " + ergebnis); 
    } 
}

Aufgabe 14 - "Kommentar III"

  • Wir wollen ein Programm schreiben, welches die Temperatur von Celsius in Fahrenheit umrechnet.
  • Erstelle dazu zwei Variablen (celsius und fahrenheit) vom Typ double und initialisiere celsius mit einem beliebigen Wert.
  • Gibt die Temperatur in Fahrenheit auf der Konsole aus.
  • Kommentiere das Programm passend!
  • Die Formel zur Umrechnung lautet:

    fahrenheit = celsius * 1.8 + 32

Aufgabe 14 - "Kommentar III"

/*
*  Name:  Max Mustermann
*  Datum: 01.05.2020
*/
class Aufgabe14 {
    public static void main(String[] args) {
        //Variable für Temperatur in Celsius 
        double celsius = 23; 
        //Variable für Temperatur in Fahrenheit 
        double fahrenheit = celsius * 1.8 + 32;
        //Ausgabe des Ergebnisses auf der Konsole 
        System.out.println("Temperatur in Fahrenheit: " + fahrenheit);
    }
}
