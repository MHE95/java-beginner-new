Ausgabe auf der Konsole

So sieht die Ausgabe auf der Konsole aus.

Aufgabe 1 - "Hello World"

  • Schreibe dein erstes, eigenes „Hello World!“ Programm! (Gebe "Hello World!" auf der Konsole aus)
    • public class NameDerKlasse{ 
    public static void main(String[] args){
        System.out.println(""); 
    } 
}

Hier muss der Text stehen, welcher Ausgegeben werden soll.

Aufgabe 1 - "Hello World"
(Lösung)

public class Aufgabe1{ 
    public static void main(String[] args){
        System.out.println("Hello World!"); 
    } 
}

Aufgabe 2 - "Ausgabe des Namens"

  • Schreibe nun ein neues Programm, das deinen Namen auf der Konsole ausgibt!
  • Die Ausgabe soll folgendermaßen aussehen:

Aufgabe 2 - "Ausgabe des Namens"
(Lösung)

public class Aufgabe2{ 
    public static void main(String[] args){
        System.out.println("Max Mustermann"); 
    } 
}

Aufgabe 3 - "Verkettung der Ausgabe"

  • Zwei Wörter können getrennt aufgeschrieben werden: "Hallo" + " Welt"
  • Ändere das Programm aus Aufgabe 2 so, dass dein Name getrennt aufgeschrieben ist!
  • Dabei soll sich die Ausgabe nicht ändern:

Aufgabe 3 - "Verkettung der Ausgabe"
(Lösung)

public class Aufgabe3{ 
    public static void main(String[] args){
        System.out.println("Max" + " Mustermann"); 
    } 
}

Aufgabe 4 - "Mehrere Zeilen Ausgeben"

  • Erweitere dein Programm, sodass in einer weiteren Zeile "Ich liebe Programmieren!" ausgegeben wird!
  • Die Ausgabe soll folgendermaßen aussehen:
  • Denkt an das Semikolon!

Aufgabe 4 - "Mehrere Zeilen Ausgeben"
(Lösung)

public class Aufgabe4{
    public static void main(String[] args){
        System.out.println("Max" + " Mustermann");
        System.out.println("Ich liebe Programmieren!");
    }
}
