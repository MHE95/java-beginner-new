Variablen

  • Speicher für Zahlen und Zeichen
  • Auf Variablen kann später im Programm zugegriffen und  könne verändert werden.
  • Es gibt viele verschiedene Datentypen, wir werden folgende drei verwenden:
    • Integer
    • Double
    • String

  • Integer können ganze Zahlen speichern, beispielsweise 2 oder 4.

Variablen - Deklaration und Initialisierung

  • Bisher haben wir Variablen in einer Zeile deklariert und initialisiert.
    • Die Deklaration ist das erstellen einer Variable.
    • Die Initialisierung ist das erstmalige Zuweisen eines Wertes zu einer Variable.



  • Deklaration und Initialisierung in einer Zeile:
    • String name = "Max Mustermann";
  • Deklaration und Initialisierung getrennt:
    • int alter;
double größe;

Variablen - Ausgabe

  • Variablen können mithilfe der Methode System.out.println() auf der Konsole ausgegeben werden.
  • Der Name der Variable muss dafür als Parameter übergeben werden (zwischen den Klammern stehen).
    • String name = "Max Mustermann";
System.out.println(name);
  • Mit "+" können mehrere Wörter und Variablen aneinandergehängt werden.
    • String name = "Max Mustermann";
int alter = 23;
System.out.println(name + " ist " + alter + " Jahre alt.");

Variablen - Ausgabe

  • Vorsicht Bei der Addition von Zahlen ohne Text!
    • int zahl1 = 4;
int zahl2 = 3;
System.out.println(zahl1 + zahl2);
  • zahl1+zahl2=4+3=7
  • Also werden die Zahlen addiert und nicht nacheinander ausgegeben.

Variablen - Ausgabe

  • Um dieses Problem zu umgehen addieren wir einen leeren Text hinzu.
    • int zahl1 = 4;
int zahl2 = 3;
System.out.println("" + zahl1 + zahl2);
  • ""+zahl1+zahl2=43
  • Die Zahlen werden nacheinander ausgegeben.
