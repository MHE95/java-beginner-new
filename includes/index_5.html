Aufgabe 10 - "Kalkulation II"

  • Schreibe ein Programm, dass den BMI berechnet.
  • Deklariere eine Integer Variablen für Körpergewicht und eine Double Variable für Körpergröße und speichere darin deine Werte.
  • Berechne mit den beiden werten deinen BMI und gebe ihn auf der Konsole aus.

  • Benutze dafür folgende Formel:

    BMI = Körpergröße in m / (Körpergewicht in kg)²

  • Hinweis:

    Zahl² = Zahl * Zahl

Aufgabe 10 - "Kalkulation II"
(Lösung)

public class Aufgabe10{ 
    public static void main(String[] args){
        double körpergewicht, körpergröße, bmi;
        körpergewicht = 85;
        körpergröße = 1.80;
        bmi =  körpergröße / (körpergewicht * körpergewicht);
        System.out.println(bmi);
    } 
}

Aufgabe 11 - "Variablentausch"

  • Dir ist beim überprüfen des Programms aus Aufgabe 10 aufgefallen, dass du in der Formel Körpergröße und Körpergewicht vertauscht hast.
  • Eigentlich müsste die Formel folgendermaßen aussehen:

    BMI = Körpergewicht in kg / (Körpergröße in m)²

  • Korrigiere nicht die Formel, sondern führe einen Variablentausch durch, um dies zu korrigieren!

Aufgabe 11 - "Variablentausch"
(Lösung)

public class Aufgabe11{ 
    public static void main(String[] args){        
        double körpergröße, körpergewicht, bmi, temporäreVariable;
        körpergewicht = 85;
        körpergröße = 1.80;
        
        temporäreVariable = körpergewicht;
        körpergewicht = körpergröße;
        körpergröße = temporäreVariable;

        bmi =  körpergröße / (körpergewicht * körpergewicht);
        System.out.println(bmi);
    } 
}
