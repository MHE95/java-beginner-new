Aufgabe 18 - "Bedingte Anweisung"

  • Schreibe ein Programm, welches nach dem Alter von 2 Personen fragt.
  • Gebe "wahr" aus falls Person 1 älter als Person 2 ist.

  • Zur Erinnerung:

    • if (Bedingung) {
    //Code der Ausgeführt werden soll, falls Bedingung erfüllt ist
}

Aufgabe 18 - "Bedingte Anweisung"
(Lösung)

import java.util.Scanner;
class Aufgabe18 {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.println("Wie alt ist Person 1?");
        int alter1 = scanner.nextInt();
        System.out.println("Wie alt ist Person 2?");
        int alter2 = scanner.nextInt();
        System.out.println("Ist Person 1 älter als Person 2?");
        if(alter1 > alter2) {
            System.out.println("wahr");
        }
    }
}

Verzweigung

  • Jetzt wollen wir zusätzlich, dass Anweisungen ausgeführt werden, falls die Bedingung nicht erfüllt ist.
  • Dafür benutzen wir zusätzlich zu dem "if" noch ein "else".
  • In dieses schreiben wir die Anweisungen, welche bei nicht erfüllter Bedingung ausgeführt werden sollen.

Verzweigung

  • Am Beispiel von vorhin sähe das folgendermaßen aus.

    System.out.println(name1);

Aufgabe 19 - "Verzweigung"

  • Ändere dein Programm aus Aufgabe 18 so, dass ausgegeben wird, welche Person älter ist.

  • Zur Erinnerung:

    • if (Bedingung) {
    //Code, der Ausgeführt werden soll, falls Bedingung erfüllt ist
} else {
    //Code, der Ausgeführt werden soll, falls Bedingung nicht erfüllt ist
}

Aufgabe 19 - "Verzweigung"
(Lösung)

import java.util.Scanner;
class Aufgabe19 {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.println("Wie alt ist Person 1?");
        int alter1 = scanner.nextInt();
        System.out.println("Wie alt ist Person 2?");
        int alter2 = scanner.nextInt();
        if(alter1 > alter2) {
            System.out.println("Person 1 ist älter.");
        } else {
            System.out.println("Person 2 ist älter.");
        }
    }
}

Verzweigung - else if

  • Manchmal reichen 2 Möglichkeiten nicht aus.
  • In unserem Beispiel gibt es noch den fall, dass beide Personen gleich alt sind.
  • Um weitere fälle hinzufügen zu können nutzen wir "else if".

Verzweigung - else if

  • Am Beispiel von vorhin sähe das folgendermaßen aus.

Vorstellung

Im Code



int alter1 = 80;
String name1 = Gertrud;
int alter2 = 10;
String name2 = Max;
if (alter1 > alter2) {
    System.out.println(name1);
} else if (alter1 < alter2) {
    System.out.println(name2);
} else {
    System.out.println(name1 + " und " + name2 
        + " sind gleich alt.");
}

Aufgabe 20 - "Mehrfach Verzweigung I"

  • Passe dein Programm aus Aufgabe 19 so an, dass auch Ausgegeben wird, wenn beide gleich alt sind.

  • Zur Erinnerung:

    • if (Bedingung1) {
    //Code, der Ausgeführt werden soll, falls Bedingung1 erfüllt ist
} else if (Bedingung2) {
    //Code, der Ausgeführt werden soll, falls Bedingung2 erfüllt ist
} else {
    //Code, der Ausgeführt werden soll, 
    //falls Bedingung1 nicht erfüllt ist und Bedingung2 nicht erfüllt ist
}

Aufgabe 20 - "Mehrfach Verzweigung I"
(Lösung)

import java.util.Scanner;
class Aufgabe20 {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.println("Wie alt ist Person 1?");
        int alter1 = scanner.nextInt();
        System.out.println("Wie alt ist Person 2?");
        int alter2 = scanner.nextInt();
        if(alter1 > alter2) {
            System.out.println("Person 1 ist älter.");
        } else if (alter1 < alter2) {
            System.out.println("Person 2 ist älter.");
        } else {
            System.out.println("Person 1 und Person 2 sind gleich alt.");
        }
    }
}

Aufgabe 22 - "Verschachtelte Verzweigung"

  • Schreibe ein Programm, dass entscheidet, ob ein Kunde ins Casino darf oder nicht.
  • Frage ihn dafür nach seinem Alter und Bargeld.
  • Man darf im Casino spielen, wenn man mindestens 21 Jahre alt ist und mindestens 100 Euro Bargeld dabei hat.
  • Nach dem Bargeld soll nur gefragt werden, wenn der Kunde über 21 Jahre alt ist.

Aufgabe 22 - "Verschachtelte Verzweigung"
(Lösung)

import java.util.Scanner;
public class Aufgabe22 {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in); 
        System.out.println("Wie alt sind Sie?");
        int alter = scanner.nextInt(); 
        if (alter >= 21) {
            System.out.println("Wie viel Bargeld haben Sie dabei?"); 
            double bargeld = scanner.nextDouble(); 
            if (bargeld >= 100) {
                System.out.println("Sie dürfen ins Casino!");
            } else {
                System.out.println("Sie müssen leider draußen bleiben.");
            }
        } else {
            System.out.println("Sie müssen leider draußen bleiben.");
        }
    }
}
