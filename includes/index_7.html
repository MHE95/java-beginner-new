Scanner

Scanner

  • Bisher mussten wird den Variablen alle Werte beim Programmieren zuweisen.
  • Jetzt wollen wir Werte (Strings und Zahlen) von der Tastatur einlesen.
  • Dafür benutzen wir einen Scanner.

Scanner - Im Code

  • Zuerst müssen wir den Scanner importieren.
  • Dafür muss folgende Zeile über der Klasse stehen:
    • import java.util.Scanner;

Scanner - Im Code

  • Dann müssen wir einen Scanner erzeugen.
  • Es wird nur ein Scanner pro Programm benötigt.
  • Dafür benötigen wir folgende Zeile:
    • Scanner meinScanner = new Scanner(System.in);

frei wählbar

Scanner - Verwendung

  • Über den Namen unseres Scanners (hier: meinScanner) können Zahlen und Texte eingelesen werden.
  • Befehl zum Einlesen von Strings (Text):
    • meinScanner.next();
  • Befehl zum Einlesen von Integer (Ganzzahl):
    • meinScanner.nextInt();
  • Eingelesen Werte sollten in Variablen gespeichert werden. Ansonsten gehen die Werte verloren.

Scanner - Beispiel

import java.util.Scanner;

public class ScannerBeispiel {
    public static void main(String[] args) {

        Scanner scanner = new Scanner(System.in); 
        System.out.println("Bitte Namen eingeben!");
    
        //Lesen eines Namens von der Tastatur 
        String name = scanner.next();

        System.out.println("Name: " + name);
    }
}

Aufgabe 15 - "Scanner I"

  • Schreibe ein Programm, welches einen Scanner erstellt.
  • Erstelle zwei passende Variablen name und alter.
  • Lies mithilfe des Scanners den Namen und das Alter des Benutzers ein.
  • Gib Namen und Alter auf der Konsole aus.

Aufgabe 15 - "Scanner I"
(Lösung)

import java.util.Scanner;
public class Aufgabe15 {
    public static void main(String[] args) { 
        Scanner scanner = new Scanner(System.in);
        String name; 
        int alter;
        name = scanner.nextLine(); 
        alter = scanner.nextInt(); 
        System.out.println("Name: " + name); 
        System.out.println("Alter: " + alter);
    }
}

Aufgabe 16 - "Scanner II"

  • Modifiziere das Programm aus Aufgabe 14 (Umrechner Celsius zu Fahrenheit), sodass die Temperatur in von der Tastatur eingelesen wird.
  • Forde den Benutzer auf die Temperatur in Celsius einzugeben.

    public static void main(String[] args) {
        double celsius = 23;

        System.out.println("Temperatur in Fahrenheit: " + fahrenheit);
    }
}

Aufgabe 16 - "Scanner II"
(Lösung)

import java.util.Scanner;
class Aufgabe16 {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.println("Bitte Temperatur in Celsius eingeben!");
        //Variable für Temperatur in Celsius 
        double celsius = scanner.nextInt(); 
        //Variable für Temperatur in Fahrenheit 
        double fahrenheit = celsius * 1.8 + 32;
        //Ausgabe des Ergebnisses auf der Konsole 
        System.out.println("Temperatur in Fahrenheit: " + fahrenheit);
    }
}
