While-Schleife

  • Dafür gibt es die While-Schleife.
  • Ein Code-Block der ausgeführt wird, solange eine Bedingung erfüllt ist.

While-Schleife Syntax

  • Im Code sieht das folgendermaßen aus:

    • while(LogischerAusdruck) {
    //Code-Block
}

  • So funkioniert sie:
    1. Der logische Ausdruck wird überprüft.
    2. Wenn der Ausdruck wahr ist, wird der Code-Block ausgeführt und das Programm springt zurück zum 1. Schritt.
    3. Wenn der Ausdruck false ist, wird die while-loop beendet.
