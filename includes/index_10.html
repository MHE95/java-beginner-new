For-Schleife Syntax

  • Im Code sieht das folgendermaßen aus:
    • for(Initialisierung; LogischerAusdruck; Update) {
    //Code-Block
}

  • So funkioniert sie:
    1. Die initialisierung wird ausgeführt.
    2. Der logische Ausdruck wird überprüft.
    3. Wenn der Ausdruck wahr ist, wird der Code-Block ausgeführt, das Update ausgeführt und das Programm springt zurück zum 2. Schritt.
    4. Wenn der Ausdruck false ist, wird die while-loop beendet.

Aufgabe 27 - "For-Schleife II"

  • Summiere die Zahlen 1 bis 10
  • Gebe das Ergebnis auf der Konsole aus.
  • Hinweis: Es wird eine Variable summe benötigt.

  • Zur Erinnerung:

    • class Aufgabe26 {
    public static void main(String[] args) {
        for (int zahl = 1; zahl < 10; zahl += 1) {
            System.out.println(zahl);
        }
    }
}

Aufgabe 27 - "For-Schleife II"
(Lösung)

class Aufgabe27 {
    public static void main(String[] args) {
        int summe = 0;
        for (int zahl = 1; zahl <= 10; zahl += 1) {
            summe += zahl;
        }
        System.out.println("Das Ergebnis ist: " + summe);
    }
}
